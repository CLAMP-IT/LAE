Uploading Authorize.net CSV file

This action must be performed if you have enabled eCheck (ACH) transactions.

  1. Login to your authorize.net account.
  2. Run a REPORT or SEARCH transactions.
  3. Select the date range if you search transactions.
  4. A button called 'Download to File' appears.
  5. After clicked the button, tick 'Include Column Headings' and select file type to be downloaded: 'Expanded Fields/Comma Separated' or 'Expanded Fields with CAVV Response/Comma Separated'.
  6. For more information click the help link on the page seen.
  7. Click Submit button.
  8. Once the file has been created, a pop up window will appear prompting you to select the location where the file should be saved.
  9. Please leave this window open until the pop up window prompting for your file location appears. You may minimize this window by clicking the _ in the upper right hand corner to continue working. Closing this window by clicking the X will cancel this action and your file will not be downloaded.
  10. Once your file has been downloaded and saved to your local machine, click the X in the upper right hand corner of this window to Close.
  11. Finally, upload this CSV file to Moodle to enrol the users making payment by echeck.
