Pop-up file or web page

This will create a pop-up window at the beginning of a lesson to a file (example: a mp3 file) or a web page. Also, a link will be printed out on every lesson page that re-opens the pop-up if necessary.

Optionally a "Close Window" button be printed at the bottom of the pop-up and the height and width of the window can be set as well.

Supported embedded file types:

  • MP3
  • Media Player
  • Quicktime
  • Realmedia
  • HTML
  • Plain Text
  • GIF
  • JPEG
  • PNG

All other file types will default to a link for download.
